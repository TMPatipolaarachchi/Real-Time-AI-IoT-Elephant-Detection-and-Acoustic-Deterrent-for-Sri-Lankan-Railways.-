# Assets Directory

This directory should contain the following assets for the app:

- `icon.png` - App icon (1024x1024px recommended)
- `splash.png` - Splash screen image (1242x2436px recommended)
- `adaptive-icon.png` - Android adaptive icon (1024x1024px)
- `favicon.png` - Web favicon (48x48px)

**Note:** For Expo Go development, these assets are optional. Expo will use default placeholders if assets are missing.

To add your own assets:
1. Place the image files in this directory
2. Ensure they match the file names specified in `app.json`
3. Run `expo start` to see your custom assets
